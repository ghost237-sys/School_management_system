M-Pesa Payments Management
M-Pesa Tools
All Transactions C2B Ledger View Log File

Use these tools to manage M-Pesa payments, including transactions paid outside the website. The ledger captures all PayBill confirmations to prevent loss during callback issues.

  • All Transactions shows STK Push and PayBill (C2B) entries with filters and quick verify.
  • C2B Ledger lists raw Safaricom confirmations (TransID, MSISDN, Bill Ref, Amount).
  • View Log File opens the raw callback log for deeper troubleshooting.
  • Verify by Reference approves or reconstructs a payment using TransID from the ledger.
